1. 预置条件：安装ibom
	Download easyeda-ibom-v0.1.2.zip
	Un-zip the downloaded archive on your hard drive.
	Go to Extension Settings dialog by using main menu Advanced -> Extensions -> Extensions Settings....
	Click Load Extension... button and add all the files in from the extracted folder using Select Files... button.
	Click Load Extension and close the Extension Settings dialog.
2. 使用方法LCEDA-->iBom-->Launch
